# SCT_WD_02
A simple and interactive Stopwatch Web Application built with HTML, CSS, and JavaScript featuring start, pause, reset, and lap tracking functionality.
⏱️ Stopwatch Web Application

A simple stopwatch built using HTML, CSS, and JavaScript. This project provides accurate time tracking with an intuitive user interface and lap recording functionality.

Features
 Start the stopwatch
 Pause the stopwatch
 Reset the timer
 Record lap times
 Real-time timer updates
 Responsive design
 
Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
